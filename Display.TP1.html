Display

La propiedad display en CSS se utiliza para controlar cómo se muestra un elemento HTML en la página. Esta propiedad especifica el tipo de caja que se utilizará para el elemento, lo que a su vez afecta su diseño y comportamiento en el flujo del documento.

Algunos de los valores más comunes de la propiedad display son:

  • display:block: El elemento se comporta como un bloque y ocupa todo el ancho disponible del contenedor padre.
  • display:inline: El elemento se comporta como una parte de texto en línea y no inicia una nueva línea.
  • display:inline-block: Combina las características de display: inline; y display: block; El elemento se comporta como un bloque, pero no inicia una nueva línea y acepta márgenes y dimensiones.
  • display:flex: Define un contenedor flexible (flex container) que organiza a sus elementos hijos (flex items) en una línea o en varias líneas, según la dirección especificada (flex-direction).
  • display:grid: Define un contenedor de cuadrícula (grid container) que organiza a sus elementos hijos (grid items) en una cuadrícula bidimensional de filas y columnas.
